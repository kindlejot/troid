Project Troid
=============

Troid is a minimalistic arcade shooter used for the _"Game Programming Essentials"_ course. This project serves as a demonstration
of professional development foundations, including Git, clear project structure, and documentation.



### System Requirements and Setup

To run and contribute to this project, please ensure you have the following tools installed.

| Requirements       | Value                              | Notes                                                  |
| ------------------ | ---------------------------------- | ------------------------------------------------------ |
| Development Engine | Unity Editor 6.2 (LTS recommended) | Note: Must match this version to ensure compatibility! |
| Scripting Language | C# (.NET Standard 2.1)             | Note: Comes with the Unity Installation                |
| Core Libraries     | Textmesh Pro plugin                | Note: For rendering text elements on UI                |
| Version Control    | Git                                | Note: **All changes must be commited via Git!**        |



### Roadmap

The project should implement the following features:
- Convert to Universal Render Pipeline
- Make Menu-structure for navigating
	- Main Menu
	- Settings Menu
	- Highscores
- Data persistency
	- Save state using PlayerPrefs
	- Highscores on a JSON file
	- Tuning with ScriptableObjects
- Add controller support
- Add Highscores (enter your initials in game over)
- Visual improvements
	- Background starfield
	- Better explosions
	- Player shield
- New dynamic Enemy
- Music
- Optimize
